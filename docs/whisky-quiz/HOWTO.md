# ウイスキーコニサー道場 HOWTO

遊び方と、家族・友人へ渡す3つの方法をまとめる。

> 注意: 問題・解説・模範解答は AI が作成し AI が監査したもの。大半は定説に沿うが、誤りが混ざる可能性はゼロではない。受験対策の正本はウイスキー文化研究所の公式テキストに置き、本アプリは反復練習の道具として使うこと。

## 1. 遊び方

### ホーム

収録問題数・挑戦済み・正答率のサマリと各モードへの入口。復習が溜まっているとバナーが出る。

### 級別演習

受験級（エキスパート／プロフェッショナル／マスター・オブ・ウイスキー／全級ミックス）と分野・問題数を選んで演習。未挑戦の問題が優先して出る。1問ごとに正誤と解説、最後に分野別の内訳が出る。下部の模擬試験は時間制限つきで、解説は最後にまとめて表示される（本番形式）。

### 世界地図

世界地図に10の産地エリアが色付きで塗られている。エリア（または下のチップ）をタップすると産地ページへ潜り、高精細な土地の地図に蒸留所ピンと産地区分（スコットランドは6区分）が現れる。

- **歴史年表をたどる**: 年代順のスクロール年表。5つの時代（起源→密造→産業→試練→復興）で色分けされ、蒸留器・樽・法律などのアイコンとともに物語をたどれる。スクロールすると上部の時代バナーが切り替わる。
- **蒸留所ピン／一覧**: タップすると各蒸留所の歴史・代表銘柄・豆知識が開く。
- **この産地の問題**: その土地に関する問題だけで演習できる（スコットランドは区分ごとにも可）。

### 官能トレーニング

- **色判定**: グラス CG の色調から色名や熟成樽を見極める。
- **色見本帳**: 12段階の色調スケール。まずここで基準を頭に入れる。
- **香味・オフフレーバー**: フレーバーホイール・香味成分・オフフレーバーの知識問題。

### 論文対策（マスター級の筆記対策）

50テーマ。設問を読み、タイマー（30/60/90分）を回して答案欄に書く。下書きは自動保存。書き終えたら「構成の骨子」「キーポイント」で自己添削し、模範解答と見比べて採点観点にチェックを入れる。

### 復習と苦手分析

- 間違えた問題は自動で復習リストに入り、正解すると外れる。
- 苦手分析には分野別正答率（低い順）と苦手キーワード（誤答率の高いタグ）が出る。「苦手克服モード」はその傾向から自動で10問を組む。

学習履歴は端末（ブラウザ）の中だけに保存される。共有端末で消したいときは苦手分析の下の「学習履歴をリセット」。

## 2. 人に渡す方法

### 方法A: 単一 HTML ファイルを渡す（最速）

`release/whisky-connoisseur.html` をそのまま送る（AirDrop・メール・LINE 等）。

- **Mac**: ダブルクリックで開くだけ。全機能が動く。
- **iPhone**: ファイル App 保存では JS が動かないことがあるため、後述の方法B（URL で開いてホーム画面へ追加）を推奨。

再生成するときはリポジトリで `python3 scripts/whisky_build.py`。

### 方法B: Web にホストして PWA として使う（iPhone に最適）

GitHub Pages への自動デプロイを同梱している（`.github/workflows/deploy-pages.yml`）。`app/` に変更を push するたびに `https://<owner>.github.io/<repo>/` へ公開される。private リポジトリで Pages が使えないプランの場合は、リポジトリを public にするか、任意の静的ホスティング（Netlify、Cloudflare Pages 等）に `app/` を置く。

1. 公開 URL を iPhone の Safari で開く。
2. 共有ボタン →「ホーム画面に追加」。アイコン付きでフルスクリーン起動し、Service Worker により**オフラインでも動く**。

URL を渡すだけで誰にでも配れる。もっとも「アプリらしい」体験。

### 方法C: App Store / TestFlight（iOS ネイティブ）

`native/` に Capacitor 製の Xcode プロジェクトを同梱している。ビルドと提出は Mac + Xcode が必須（本リポジトリの CI 環境では署名・提出はできない）。

前提: [Apple Developer Program](https://developer.apple.com/jp/programs/)（年 99 USD）への加入。

```bash
# Mac で一度だけ
cd native
npm install
python3 ../scripts/whisky_build.py   # www へ最新データを同期
npx cap sync ios
npx cap open ios                     # Xcode が開く
```

Xcode での手順:

1. `App` ターゲット → Signing & Capabilities で自分の Team を選ぶ。
2. Bundle Identifier `com.whiskydojo.app` を自分のドメイン形式（例 `jp.yourname.whiskydojo`）へ変更。
3. 実機を選んで ▶ で動作確認。
4. Product → Archive → **Distribute App**。
   - 家族に渡すだけなら **TestFlight** が簡単: App Store Connect でアプリを登録 → 内部テスターに妻の Apple ID を追加 → 妻の iPhone に TestFlight アプリを入れて招待を受ける。審査は簡易で数分〜数時間。
   - 一般公開するなら App Store 審査へ提出（スクリーンショット・説明文・プライバシー表記が必要。本アプリは通信・個人データ収集なしなので「データ収集なし」で申告できる）。

アプリアイコンは `app/icons/icon-1024.png` を Assets.xcassets に設定済み。

Mac 向けは、Apple Silicon Mac なら iOS アプリがそのまま動く（App Store Connect の「Mac で利用可能」を有効化）。加えて方法A/Bで Mac は常にカバーされる。

## 3. データを増やす・直す

- 問題の原本は `app/data/raw/questions/*.jsonl`（1行1問）。行を追加・修正したら `python3 scripts/whisky_build.py` で反映。
- 論文テーマは `app/data/raw/essays/*.jsonl`、地図は `app/data/raw/map.json`、色見本は `app/data/raw/sensory.json`。
- スキーマは `docs/whisky-quiz/design.md` を参照。ビルド時に検証され、不正な行は警告つきで除外される。
