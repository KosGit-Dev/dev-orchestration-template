# ウイスキーコニサー道場 要件定義

ウイスキー文化研究所「ウイスキーコニサー資格認定試験」（ウイスキーエキスパート／ウイスキープロフェッショナル／マスター・オブ・ウイスキー）の合格を目指す学習ゲーム。本人と、アプリを渡された家族・友人がひとりで学習を完結できることを狙う。

## 利用者と利用環境

- 発案者本人（受験者）と、その家族（ウイスキー愛好家。試験勉強ではなく娯楽として遊ぶ）。
- iPhone と Mac。最優先は Web（ブラウザ／PWA）、並行して App Store 提出可能な iOS アプリを用意する。
- オフラインでも使える（PWA キャッシュ、または単一 HTML ファイル）。

## 非機能要件

- 外部サービスへ一切通信しない（自己完結。学習履歴は端末内のみ。個人情報を扱わない）。
- ビルド不要で動く素の HTML/CSS/JS。依存ライブラリなし。
- iPhone SE 級の画面幅（375px）から Mac のブラウザまでレイアウトが崩れない。
- 出題データは生成時と統合時の二段で検証する（JSON 妥当性、選択肢4件、answer 範囲、重複除去）。

## 問題データの品質方針

- 出題は定説・確立された事実に限る。2020年以降に変動しうる時事（生産量順位、蒸留所の開閉、受賞）は避ける。
- 生成エージェント（sonnet/opus）の出力を、独立の監査エージェント（opus）が事実確認し、疑わしい問題は修正または削除する。
- それでも誤りが残りうる旨をアプリ配布時の注意書きに明記する（資格対策の正本は公式テキストである）。
